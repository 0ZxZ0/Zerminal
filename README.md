# Zerminal 

Zerminal is a lightweight, custom command-line interface (CLI) environment written in Python. It allows users to navigate the file system, manage files and directories, and view system information through simple, intuitive commands. 

This project was created to practice Python development, specifically working with the `os` and `shutil` modules.

## Commands

### cm  - create map (folder)
### cf  - create file
### sf  - show file (map name)0
### saf - show all folders (Warning: scans whole drive)
### rm  - remove [name]
### update - update a zerminal0
### nup - over update
### si - system info
### read - read text file0
### exit - close terminal
### vlc -  vlc (video name)
